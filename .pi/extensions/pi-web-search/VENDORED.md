# Vendored pi-web-search

This directory contains the source distributed as `pi-web-search` version
1.3.1. It is bundled so Open Law Lens users do not need a separate project-local
Pi package installation.

- Upstream author: `ttttmr`
- Upstream repository: <https://github.com/ttttmr/pi-web-search>
- Package: <https://www.npmjs.com/package/pi-web-search/v/1.3.1>
- Declared license: MIT

The upstream npm package and repository did not include a standalone license
file when this version was bundled. `LICENSE` records the standard MIT terms
declared by the package metadata and attributes the package author. Trailing
whitespace in the upstream TypeScript files was normalized for this repository.
